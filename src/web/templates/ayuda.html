Guía rápida para ejecutar en macOS + Safari.

Pasos

  1. Crear y activar venv: python3 -m venv .venv && source .venv/bin/activate
  2. Instalar: pip install flask graphviz
  3. Instalar Graphviz: brew install graphviz  →  verificar: dot -V
  4. Levantar: flask --app src.web.app run -p 8000 y abrir: open -a "Safari" http://127.0.0.1:8000
